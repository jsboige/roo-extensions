# Terminal et Environnement

## PowerShell

Ton terminal est un PowerShell. Regles :
- Utilise `pwsh -c "..."` pour les commandes
- Le terminal peut etre non-persistant : prefixe tes commandes avec l'activation d'environnement si necessaire
- En PowerShell, utilise `;` au lieu de `&&` pour enchainer les commandes
- Ne lance pas de longues commandes de traitement sans avoir verifie leur impact sur de petits echantillons

## Preference MCPs

Utilise de preference les MCPs disponibles qui sont generalement plus rapides et accessibles que les commandes de terminal et les outils natifs. Par exemple utilise Quickfiles pour l'exploration de repertoires et la lecture batch de fichiers.
